# Explanation

- **Definition**
    - a standalone third party library
    - responsible for managing _global state_ from one accessible store to update using actions
- similar to [Context API](Context%20API.md)+[useReducer](useReducer.md) working together
- [Redux & React connection](Redux%20&%20React%20connection.md)
- Used [Action Creators](Action%20Creators.md) instead of typing action manually like in UseReducer
